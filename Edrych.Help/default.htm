Welcome to Edrych

Edrych is a data explorer for a variety of database servers. When dealing with multiple different database servers, one typically has to use a different tool for each server. Now, you can browse all you want on Edrych, while executing queries that can actually talk between servers! No more need for setting up clunky linked servers, because Edrych handles it all for you!

Understanding Edrych

There are a couple of components to Edrych that you should learn how to navigate to provide for the best experience. When you first open up Edrych, you get prompted by the Connection Dialog. Once you connect to a server (or cancel out), you'll see the main window:



This screenshot shows after you've opened up a Query, but you can easily find that by going to File->New, or clicking the New button ().

On the left is the Server Browser, which allows you to easily look through a database server that you've connected to and discover its structure. Use this to navigate through the server and help you formulate your queries.

On the right is the Query Window, which allows you to run queries against the database servers. It's entirely free-form, so use caution! The database server will do whatever you tell it to do with your credentials!
