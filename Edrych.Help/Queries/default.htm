Queries

The most real estate of Edrych is taken up by the beloved query window. This is where it all comes together--where you can run free-range queries against eagerly awaiting databases and get results instantly. Not only can you get results from select queries, but you also get confirmation from data alteration operations through the Messages tab.

But what makes Edrych special is the ability to use Interserver Queries, or queries between two different database servers, without the need for linked servers. Edrych handles it all for you, but you've got to use some special syntax to get it to work properly.

Make sure you read about Running a Query so you know what query is running when! And read Switching Databases, too, so you know how to change your active connection!
