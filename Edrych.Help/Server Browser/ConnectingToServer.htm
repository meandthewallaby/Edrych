Connecting to a Server

When first opening Edrych, the Connection Dialog will pop up to instantiate your session connected to the database server of your choice. In order to connect to different servers, there are two ways you can connect to another server.

The first is to connect through the File menu. Simply go to File-->Connect, and it will pop up the Connection Dialog for you to add a server:



The second is to connect by hitting the Connect button just above the Server Browser:
